# Mobile Sales Analysis Dashboard

# Overview

This project analyzes mobile sales data using SQL, Python, and Power BI to identify sales trends, customer behavior, and brand performance.

# Tools & Technologies

* SQL
* Python (Pandas, NumPy)
* Power BI

# Project Workflow

1. Data Cleaning and Preparation
2. SQL Data Analysis
3. Exploratory Data Analysis using Python
4. Dashboard Creation in Power BI

# Key Insights

* Identified top-performing mobile brands and models
* Analyzed customer purchasing patterns and payment methods
* Compared sales performance across cities
* Observed overall sales and transaction trends
